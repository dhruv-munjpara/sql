🧩 TASKS

✅ TASK 1: INNER JOIN
Task:
Display student name, course name, and enrollment date.
📌 Concept: INNER JOIN
SELECT students.name,courses.course_name ,enrollments.enroll_date FROM enrollments  INNER JOIN students  on enrollments.student_id=students.student_id INNER JOIN courses  on enrollments.course_id=courses.course_id;


✅ TASK 2: LEFT JOIN
Task:
Display all students with their enrolled course names.
Show NULL if a student is not enrolled.
📌 Concept: LEFT JOIN
SELECT students.name,courses.course_name FROM students LEFT JOIN enrollments on students.student_id=enrollments.student_id LEFT JOIN courses on enrollments.course_id=courses.course_id;

✅ TASK 3: RIGHT JOIN
Task:
Display all courses and students enrolled in them.
📌 Concept: RIGHT JOIN
SELECT courses.course_name,students.name,enrollments.enroll_id
FROM students 
RIGHT JOIN enrollments ON students.student_id = enrollments.student_id RIGHT JOIN courses ON enrollments.course_id = courses.course_id;


✅ TASK 4: JOIN + WHERE
Task:
Display students who are enrolled in the Java course.
📌 Concept: JOIN + WHERE
SELECT students.name FROM students INNER JOIN enrollments on students.student_id=enrollments.student_id INNER JOIN courses ON enrollments.course_id=courses.course_id where  courses.course_name="java";





✅ TASK 5: CREATE VIEW Task:
 Create a view named student_course_view that shows student name, city, course name, and fees.
